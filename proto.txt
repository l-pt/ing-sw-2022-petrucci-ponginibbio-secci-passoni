Descrizione del protocollo

Legenda
S: Server
C: Client

Prima della partita (Lobby)
C -> S apertura connessione TCP
S -> C AskUsernameMessage
C -> S SetUsernameMessage

Se il client è il primo connesso, il server chiede anche numero massimo di
player e modalità esperto, altrimenti questa sezione è saltata:
S -> C AskPlayerNumberMessage
C -> S SetPlayerNumberMessage
S -> C AskExpertMessage
C -> S SetExpertMessage
